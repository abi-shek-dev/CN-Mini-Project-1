# CN-Mini-Project-1

## Overview
CN-Mini-Project-1 is a Python-based mini project focused on cryptocurrency price tracking and visualization. This project allows users to collect, log, and visualize cryptocurrency prices in real-time.

## Features
- **Cryptocurrency Price Data Collection**: Gather price data using the `crpt.py` module.
- **Data Logging**: Store collected data in a CSV file format (`crypto_prices_log.csv`).
- **Dashboard Interface**: Visualize cryptocurrency prices and market information through a dashboard interface (`dashboard.py`).

## Technology Stack
- **Programming Language**: Python
- **Data Storage**: CSV files
- **Dashboard Framework**: (Specific framework not detailed)

## Installation
To set up the project locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/abi-shek-dev/CN-Mini-Project-1.git
   ```
2. Navigate to the project directory:
   ```bash
   cd CN-Mini-Project-1
   ```
3. Install the required dependencies (if any):
   ```bash
   pip install -r requirements.txt
   ```

## Usage
To run the cryptocurrency price collection and visualization, execute the following command:
```bash
python dashboard.py
```
